# 📱 Rolodex App

A Flutter application that replicates an iOS-style contacts experience, allowing users to browse and explore contact lists grouped alphabetically.

---

## 🚀 Description

**Rolodex** is a Flutter app built as part of an advanced UI learning pathway. It demonstrates how to manage structured data, build adaptive layouts, and create platform-consistent interfaces using Cupertino widgets.

The app displays contact groups and allows users to navigate into detailed views where contacts are organized alphabetically, similar to the native iPhone Contacts app.

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/rolodex.git
```

2. Navigate into the project:

```bash
cd rolodex
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the app:

```bash
flutter run
```
